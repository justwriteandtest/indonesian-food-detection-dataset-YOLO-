# Indonesian Food Detection Dataset for YOLO (Small)

## About

This dataset is meant to train YOLO models, containing 1250 images of various quality. The images were taken from the validation and test subsets of [the Indonesian Food Dataset at Kaggle](https://www.kaggle.com/datasets/rizkyyk/dataset-food-classification), and annotated with Roboflow's assistance.

As per the source dataset, this dataset contains 13 classes:
- **Ayam**, mostly consisted of fried chicken. Ocassionally contains roasted chicken images.
- **Burger** (some images also contain French Fries, annotated in its own class below).
- **French Fries**
- **Gado-gado**
- **Ikan**, mostly consisted of fried fish. Also contains images of roasted or steamed fish.
- **Mie Goreng**, which also contains mie ayam images.
- **Nasi Goreng**
- **Nasi Padang**, which comes in both plate and rarely, buffet-esque servings.
- **Pizza**
- **Rawon**
- **Rendang**
- **Sate**
- **Soto**

To use this dataset, you can simply clone this repository.
